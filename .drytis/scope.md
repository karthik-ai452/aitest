# Scope

- Side navigation with menu items: Dashboard, Test Suites, Test Runs, Agents, Users/Team, Settings, Documentation
- Top header with search bar, notifications, help, Execute Run button, user avatar
- Main content: Test Runs table with columns (ID, Suite Name, Status, Environment, Duration, Time)
- Pagination footer
- Dark theme with custom color tokens
- Hover effects and transitions

## Out of Scope

- User authentication
- Real-time data updates
- Interactive filtering / sorting
- Mobile responsive layout
- Multiple pages / routing
