# AI Tester — CmdRunner QA Dashboard

## Overview
A dark-themed QA Engineering Dashboard called **CmdRunner** for managing and viewing automated test runs. Built with plain HTML, Tailwind CSS (CDN), and Google Fonts.

## Tech Stack
- **HTML5** — single-page static site
- **Tailwind CSS** — via CDN with custom dark theme config
- **Google Fonts** — Geist, JetBrains Mono, Material Symbols Outlined
- **No build step** — served as static files

## Key Decisions
- Single `index.html` file with embedded Tailwind config
- Custom color palette (Material Design 3 inspired dark theme)
- Responsive sidebar + top nav + data table layout
- Status badges: Passed (green), Failed (red), Warning (amber)
